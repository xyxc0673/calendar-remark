## Features

- [x] Custom remark to a date
- [x] Lunar calendar
- [x] Holiday and Festival
- [x] Year progress
- [x] Dark mode
- [x] Responsive design

## Tech Stack

- [x] React
- [x] TypeScript
- [x] Tailwind CSS
- [x] Vite
- [x] Jotai

## Development

```bash
# install dependencies
$ pnpm install

# serve with hot reload at localhost:5173
$ pnpm dev

# build for production and launch server
$ pnpm build
$ pnpm start
```
